# Dismas Downloader — Архитектура и задачи

## Атмосфера
- Вдохновлено Darkest Dungeon, особенно образом Дисмаса: безысходность, стресс, ржавая сталь, слабая надежда
- Интерфейс должен быть тяжёлым, готическим, с анимациями и визуальными эффектами DD

## Технологии
- Electron (UI, main process)
- Rust (через NAPI-RS, sidecar для yt-dlp)
- yt-dlp/ffmpeg (бинарники)
- Темы: CSS/IMG/Fonts + JSON-инструкция, хранятся в themes/*

## Основные задачи
1. **Архитектура приложения**
    - Electron main process: управление окнами, IPC, загрузка тем, запуск Rust sidecar
    - Preload: contextBridge для безопасного API
    - Renderer: UI, загрузка темы, взаимодействие с пользователем
    - Rust sidecar: обёртка над yt-dlp, прогресс, ошибки, возврат статуса
2. **Система тем**
    - Темы лежат в themes/* (root)
    - Структура темы: theme.json, styles.css, assets/
    - Встроенный конструктор тем: создание, экспорт, импорт
    - Возможность смены темы на лету
3. **UI/UX**
    - Готическая палитра, шрифты Mason/Goudy, эффекты (факел, пыль, кровь)
    - Все статусы и сообщения — в стиле DD
    - Кастомизация через темы
4. **Функционал загрузчика**
    - Ввод ссылки, выбор пути сохранения
    - Прогресс-бар, статус, лог ошибок
    - Логирование загрузок (SQLite)
    - Автообновление yt-dlp
5. **Безопасность и структура**
    - Нет nodeIntegration в renderer
    - Все пути через app.getPath/process.resourcesPath
    - IPC только через preload

## Минимальный MVP
- Скачивание видео по ссылке
- Темная тема DD по умолчанию
- Смена темы через UI
- Лог загрузок

## Расширения
- Конструктор тем
- Галерея тем
- Импорт/экспорт тем
- Расширенные настройки загрузки
